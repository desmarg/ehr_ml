ehr_ml.utils¶

Various utilities useful for performing analysis on EHR data

class ehr_ml.utils.OnlineStatistics(old_data: Optional[Dict[str, Any]] = None)¶

A class for computing online statistics such as mean and variance. From https://en.wikipedia.org/wiki/Algorithms_for_calculating_variance.

Initialize online statistics. Optionally takes the results of self.to_dict() to initialize from old data.

add(newValue: float) → None¶

Add an observation to the calculation.

mean() → float¶

Return the current mean.

standard_deviation() → float¶

Return the current standard devation.

to_dict() → Dict[str, Any]¶

Serialize the data to a dictionary that can be encoded with JSON. Feed the resulting dictionary back into the constructor of this class to extract information from it.
