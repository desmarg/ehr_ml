ehr_ml.ontology¶

ehr_ml.ontology is a module for handling relationships between observation codes.

The fundamental ontological structure is in the form of a rooted network. There is a root node at the top that contains multiple children and each child in turn contains children. Nodes can have multiple parents.

Note that the ontologies operate in a different code space from the raw timeline codes and thus have a seperate TermDictionary.

Accessing this ontology is done through the OntologyReader class.
